# Personal Space Art Direction

This document defines the current visual baseline for generated or commissioned 2D personal space assets in Orbit.

Use this as the default style guide for:

- furniture props
- personal space object sprites
- window-view inserts
- portrait overlays
- scene concept references

Unless a specific scene intentionally needs a different mood or camera language, new assets should stay within this baseline.

## Visual Baseline

- lighting: top-left
- angle: 3/4 view, around 25 degrees
- shadow: soft bottom-right
- highlight: edge-only, not full-surface gloss
- material: simplified layered shading
- style: semi-real stylized

## Intent

This baseline exists to keep rental, office, and estate assets visually coherent even when they are generated across different sessions or by different agents.

The target feel is:

- readable on mobile
- cohesive across mixed props
- more grounded than cartoon art
- lighter and cleaner than realistic rendering

## Avoid

- fully realistic rendering
- cartoon exaggeration
- inconsistent light direction between props
- centered or flat front-view furniture unless the scene explicitly calls for it
- heavy glossy highlights across full surfaces
- hard black shadows

## Notes For Future Expansion

- 2D props, portraits, and window views should continue to use this baseline unless a future 3D-ready scene pack defines its own variant
- if a new scene family intentionally changes the look, document that as a named variant instead of silently drifting from this baseline
- this document is a style consistency reference, not a runtime dependency
